# Getting Started for Store Owners

Purpose: Help a store owner log in, understand navigation, and configure essentials.

## Overview
Brief tour of the app layout and primary modules.

## Prerequisites
- Store owner account and login URL
- Browser: Chrome recommended

## Sign In
1. Go to the login page.
2. Enter your email and password.
3. Click Login.

![Login page](../_images/store-owner/getting-started-01.png)

## Navigation Basics
- App header and main nav
- Where to find Dashboard, Inventory, Trade‑Ins, Rewards, Reporting

![Navigation](../_images/store-owner/getting-started-02.png)

## Initial Configuration Checklist
- Store profile & branding
- Taxes & time zone
- User roles (owner/manager/clerk)

## Tips
- Use bookmarks for quick access

## FAQs
- I forgot my password
- I don’t see a module I expect
