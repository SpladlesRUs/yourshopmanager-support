# Trade-Ins Workflow

Purpose: Intake, valuation rules, approval, and receipt.

## Overview
Automate buy offers with flexible rules.

## Start a Trade‑In
1. Go to Trade‑Ins → New
2. Scan/search item; set condition
3. System suggests value via rules; adjust if needed

![Start Trade‑In](../_images/store-owner/tradeins-01.png)

## Review & Approve
- Review totals, store credit vs cash
- Capture customer signature (if applicable)

## Complete & Receipt
- Print/email receipt
- Credit balance updated

- Overriding rules safely
