# Rewards Program Setup (Shillings)

Purpose: Configure earn/redemption rules and manage balances.

## Overview
Increase repeat visits with a simple points program.

## Configure Rewards
1. Go to Rewards → Settings
2. Set earn rate (e.g., 1 point per $1)
3. Set redemption value and limits

![Rewards Settings](../_images/store-owner/rewards-01.png)

## Manage Balances
- Customer profile → Rewards tab
